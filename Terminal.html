What are flags?

Flags modify how a program behaves. For example, when you pass a -a flag to ls, it will also include hidden files, A.K.A dotfiles since they start with dots

Alternatives to common commands

Open source is beautiful in that you can try out alternatives to the commands that we just learned about Some add more functionality, and some are more performant.

  • fd
    • An alternative to the find command. Very handy when you can’t find a certain file
  • rg
    • Stands for rip grep, which looks for patterns within files so if you can’t remember which file has the keyword “hello world”, it’ll tell you where it’s located and which line in the file it’s located.
